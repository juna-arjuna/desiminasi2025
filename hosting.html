Hosting

Bagian ini menjelaskan bagaimana menyimpan dan menayangkan hasil coding secara online melalui layanan hosting gratis seperti GitHub Pages.

1. Apa itu Hosting?

Hosting adalah layanan yang menyediakan ruang di server (komputer khusus) untuk menyimpan file-file website Anda, seperti file HTML, gambar, video, dan lain-lain, agar dapat diakses melalui internet. Jika Anda membuat halaman HTML secara offline (di komputer lokal), maka hanya Anda yang bisa melihatnya. Namun, dengan hosting, file HTML tersebut bisa dibuka oleh siapa pun melalui browser dari mana saja, karena sudah tersimpan di server online.

2. Hosting Gratis

Hosting gratis adalah layanan penyimpanan website di internet yang disediakan tanpa biaya, memungkinkan pengguna untuk mengunggah dan membagikan halaman web seperti file HTML tanpa harus membeli server sendiri. Layanan ini sangat cocok bagi pemula, pelajar, atau siapa pun yang ingin belajar membuat website tanpa mengeluarkan uang. Beberapa contoh layanan hosting gratis yang populer antara lain : GitHub Pages, 000webhost, dan Geocities.ws, yang menawarkan kemudahan dalam mengunggah file HTML serta memberikan alamat website sendiri. Meskipun biasanya memiliki keterbatasan seperti ruang penyimpanan kecil, kecepatan akses terbatas, dan alamat domain yang panjang, hosting gratis tetap menjadi solusi praktis untuk mengubah halaman HTML offline menjadi website yang bisa diakses secara online oleh siapa saja.

3. Mendaftar di Layanan Hosting github.com

Untuk membuat website online secara gratis, Anda bisa mendaftar di layanan hosting gratis seperti https://github.com. Proses pendaftarannya cukup mudah dan tidak memerlukan biaya apa pun. Anda hanya perlu mengunjungi situs tersebut, lalu klik menu “Sign Up” atau “Register”, dengan langkah sebagai berikut :

1. Buat Akun GitHub

- Buka https://github.com

- Klik Sign up

- Masukkan:

Username → misalnya: websiteku

Email aktif :

Password :

- Verifikasi akun lewat email dari GitHub

- Setelah login, Anda akan masuk ke dashboard GitHub.

2. Buat Repository Baru

- Klik tombol “+” di pojok kanan atas → pilih New repository

- Isi form :

Repository name → misalnya web-pembelajaran

(opsional) beri deskripsi

- Klik Create repository

📦 Sekarang Anda punya wadah online untuk menyimpan file HTML-nya.

3. Upload File HTML dan CSS

- Di halaman repository, klik tombol “Add file” → “Upload files”

- Drag & drop semua file web Kak (misalnya index.html, style.css, gambar, dsb)

- Setelah semua muncul di daftar file, klik Commit changes (di bawah halaman)

→ File langsung tersimpan online di GitHub.

4. Aktifkan GitHub Pages

Di repository, klik menu Settings (paling kanan di atas)

Scroll ke bawah → cari bagian “Pages”

Di bagian Source, pilih :

Branch → main

Folder → / (root)

Klik Save

💬 Tunggu beberapa detik sampai GitHub memproses (kadang 1–2 menit)

5. Lihat Alamat Website-nya

Setelah aktif, di bagian atas “Pages” akan muncul pesan seperti:

Your site is live at

👉 https://websiteku.github.io/web-pembelajaran/

🎉 Klik link itu — dan halaman tekateki.html anda sudah online dan bisa dibuka siapa saja di seluruh dunia!
