1. Apa itu Coding?

Coding adalah proses menulis instruksi dalam bentuk kode atau bahasa pemrograman yang dapat dipahami oleh komputer. Melalui coding, seseorang memberi tahu komputer apa yang harus dilakukan — mulai dari menjalankan perhitungan, menampilkan teks dan gambar, hingga membuat aplikasi, game, atau situs web. Coding ibarat bahasa komunikasi antara manusia dan mesin.Dalam praktiknya, coding dilakukan dengan menggunakan bahasa-bahasa khusus seperti Python, JavaScript, HTML, CSS, C++, dan lainnya, tergantung pada tujuan yang ingin dicapai. Misalnya, HTML digunakan untuk membuat struktur halaman web, sedangkan Python sering digunakan untuk analisis data, kecerdasan buatan, atau pengembangan aplikasi. Orang yang menulis kode disebut programmer atau developer. Belajar coding sangat penting di era digital karena membuka peluang untuk menciptakan teknologi, memecahkan masalah, dan mengembangkan solusi otomatisasi. Bahkan kini, dengan bantuan AI, proses belajar dan menulis kode menjadi jauh lebih mudah dan bisa diakses siapa saja.

2. Apa bahasa HTML?

HTML (HyperText Markup Language) adalah bahasa dasar dalam pembuatan halaman web. HTML digunakan untuk menyusun struktur konten seperti teks, gambar, tabel, tautan, dan elemen-elemen lainnya agar dapat ditampilkan secara teratur dalam browser. HTML bersifat statis, namun sangat mudah dipelajari dan sering digabungkan dengan CSS dan JavaScript untuk membuat tampilan dan interaksi yang lebih dinamis pada situs web. Setiap elemen HTML ditandai dengan tag.

3. Guru dan Kode HTML Buatan Kecerdasan Artifisial

Guru atau pengajar kini dapat dengan mudah membuat media pembelajaran interaktif menggunakan coding HTML berbasis AI, hanya dengan memberikan perintah atau prompt dalam bahasa alami kepada sistem kecerdasan buatan seperti ChatGPT. Dengan bantuan AI, pembuatan halaman web yang berisi teks, gambar, video, soal interaktif, atau simulasi dapat dilakukan tanpa harus menguasai seluruh sintaks HTML secara manual. Cukup dengan mengetikkan instruksi seperti “Buat halaman HTML tentang sistem tata surya dengan gambar dan deskripsi tiap planet,” AI akan menghasilkan kode siap pakai yang bisa langsung digunakan atau dimodifikasi sesuai kebutuhan. Pendekatan ini sangat membantu dalam mempercepat proses pembuatan bahan ajar digital dan meningkatkan kreativitas guru dalam merancang pembelajaran berbasis teknologi.

Berikut adalah contoh prompt yang bisa kita perintahkan pada chatGPT untuk membuatkan coding HTML

Buatkan halaman website teka-teki dengan program HTML. Tuliskan judul: TEKA-TEKI pada bagian atas. Teki-tekinya Adalah : "Aku adalah seekor hewan. Jumlah kakiku ada empat buah. Aku memiliki ekor. Aku juga memiliki belalai. Siapakah aku?" Buatkan kotak untuk menjawabnya. Jawabannya adalah: Gajah atau gajah atau GAJAH. Jika menjawab benar maka beri feedback "Jawaban Anda benar, hebat". Jika jawabannya salah maka beri feedback: "Jawaban Anda salah, silahkan coba lagi."
