1. Menjalankan HTML di Browser

Untuk membuat coding HTML menjadi file yang bisa dijalankan di browser, langkahnya sangat mudah dan dapat dilakukan oleh siapa saja. Pertama, buka program teks editor yang tersedia di komputer, seperti Notepad, Notepad++, Visual Studio Code, atau Sublime Text. Setelah itu, ketik atau tempelkan kode HTML yang telah kamu buat ke dalam teks editor tersebut. Pastikan seluruh bagian kode sudah benar dan lengkap, misalnya terdiri dari elemen dasar seperti tag html, tag head, dan tag body.

2. Notepad

Notepad adalah sebuah aplikasi bawaan sistem operasi Windows yang berfungsi sebagai editor teks sederhana. Aplikasi ini sudah tersedia secara otomatis di hampir semua komputer atau laptop yang menggunakan Windows, sehingga pengguna tidak perlu mengunduh atau menginstalnya secara terpisah. Notepad digunakan untuk menulis dan mengedit teks polos, tanpa format tambahan seperti huruf tebal, miring, atau ukuran huruf. Karena tampilannya yang sederhana dan ringan dijalankan, Notepad sering dimanfaatkan untuk mencatat hal-hal singkat atau menulis kode pemrograman dasar seperti HTML, CSS, JavaScript, dan bahasa pemrograman lainnya.

Untuk membuka Notepad, pengguna cukup menekan tombol Start atau menekan tombol Windows di keyboard, lalu mengetikkan kata "Notepad" di kolom pencarian. Setelah itu, klik ikon Notepad yang muncul di daftar hasil pencarian. Meskipun fitur yang tersedia sangat dasar, Notepad sangat cocok digunakan oleh pemula yang ingin mulai belajar membuat halaman web atau mengedit file teks. Bagi pengguna yang membutuhkan fitur tambahan seperti pewarnaan kode (syntax highlighting), penomoran baris, atau pelengkap otomatis saat menulis kode, disarankan untuk menggunakan editor yang lebih canggih seperti Notepad++ atau Visual Studio Code.

Berikut ini adalah coding html tentang "teka-teki" yang telah dibuatkan oleh chatGPT di-copy dan di-paste-kan pada Notepad

Setelah selesai menulis kode, simpan file tersebut dengan cara klik menu “File”, lalu pilih “Save As”. Beri nama file tersebut dengan akhiran .html, misalnya teka-teki.html. Pada bagian “Save as type”, pilih “All Files” agar sistem tidak menyimpannya sebagai file teks biasa. Pastikan juga encoding-nya menggunakan UTF-8 untuk mendukung karakter Indonesia. Kemudian klik “Save”.

3. Double Click File HTML

Langkah terakhir adalah menjalankan file tersebut. Buka folder tempat kamu menyimpan file HTML tadi (misal sebagai contoh nama filenya adalah: teka-teki.html, lalu klik dua kali pada file tersebut. File akan terbuka secara otomatis di browser seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge. Halaman HTML kamu kini bisa dilihat dan dijalankan layaknya sebuah website. Dengan cara ini, kamu bisa membuat dan menguji halaman web secara offline sebelum diunggah ke internet.
